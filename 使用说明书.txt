Q: 双击没反应？
A: 右键「启动GUI.bat」→ 编辑 → 另存为
   编码选 ANSI → 保存。再双击。

Q: 什么是 API Key？
A: 就是 AI 的钥匙。
   去 https://platform.deepseek.com 注册
   → API Keys → 创建 → 复制粘贴到软件里

Q: 文件处理后在哪？
A: 就在你选的 PDF 文件同目录下。
